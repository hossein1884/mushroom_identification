This code is written with a **Decision Tree Algorithm** and, with input data in `.csv` format on the type of mushroom, can guess whether the mushroom is poisonous or non-poisonous based on its appearance and shape.


## Installation
You just need to install python language from [python.org](https://www.python.org/) and the **NumPy** and **MatplotLib** libraries with the following commands after creating a **Virtual Environment** in the terminal...
```
python -m venv .venv
```
```
pip install numpy
```
```
pip install matplotlib
```
Finally, you need to install **Jupyter Notebook** in whatever IDE you have.
